# Fake-Review-Detector
A machine learning project that detects fake/spam reviews using 
NLP text features and behavioral signals.

## 📌 Problem Statement
Fake reviews mislead consumers and damage honest businesses.
This project builds a classifier to automatically flag suspicious reviews.

## 🤖 Models Used
- Logistic Regression
- Naive Bayes
- Random Forest ✅ (best performer)
- Gradient Boosting

## 📂 Dataset
Kaggle — Fake Reviews Dataset (40,000 reviews, 50% fake / 50% real)
